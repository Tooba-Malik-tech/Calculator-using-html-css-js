# Responsive Calculator

A sleek and fully responsive calculator built with **HTML**, **CSS**, and **JavaScript**. It supports basic arithmetic operations and adapts smoothly to all screen sizes, from desktops to mobile devices.

## Features

- Addition, subtraction, multiplication, and division
- Responsive design for all devices
- Clear (C) and delete (⌫) functionalities
- Smooth button animations and transitions
- Lightweight and easy to customize

## Demo

You can try the calculator by opening `index.html` in your browser.

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/responsive-calculator.git
````

2. Open `index.html` in any modern web browser.
3. Use the buttons or keyboard to perform calculations.

## Technologies Used

* HTML5
* CSS3 (Flexbox/Grid for responsiveness)
* Vanilla JavaScript
